# 둥가리 코퍼레이션 홈페이지

AI 기반 애니메이션 뮤직비디오 제작 스튜디오 — 둥가리 코퍼레이션의 온보딩 페이지입니다.

## 기술 스택

- React + Vite
- Tailwind CSS v4
- GitHub Pages (정적 배포)

## 로컬 실행

```bash
npm install
npm run dev
```

## 배포

`main` 브랜치에 push하면 GitHub Actions를 통해 GitHub Pages로 자동 배포됩니다.

> Settings → Pages → Source를 **GitHub Actions**으로 설정해야 합니다.
